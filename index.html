Description:
This project uses two grid boxes to find the shortest path between them. To find the shortest path, I took the bottom right corner of the green box and subsequent boxes and the top left corner of the red box/target box. My algorithm determines the possible moves to take and finds the best move to take using pixel distances between the current box and target box.

Instructions:
Please pick any box that is not the green box. The box you pick will be colored red. After clicking, a path will be shown from the green box to your box showing the shortest possible path that can be taken. To pick another box, refresh the page.
